# Movie Explorer Dashboard 

A modern, fully responsive movie browsing application built with **HTML, CSS, and Vanilla JavaScript**. This project uses the **TMDB API** to fetch real-time movie data and display it in an interactive interface.

## Features 
- Dark themed modern UI
- Responsive movie card grid
- Real-time movie search
- Movie details modal view
- Dynamic API data fetching
- Library-free implementation
- Client-side JavaScript functionality

## Technologies Used 
- HTML5
- CSS3
- JavaScript (ES6)
- TMDB API


## Purpose
This project demonstrates API integration, responsive web design, and dynamic content rendering using pure JavaScript without external libraries.
